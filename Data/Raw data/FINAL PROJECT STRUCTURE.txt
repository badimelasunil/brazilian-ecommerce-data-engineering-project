WHAT EACH FOLDER DOES
data/raw/

Contains original Kaggle datasets.

Never modify these files.

data/cleaned/

Contains cleaned datasets after:

null handling
datatype fixing
duplicate removal
data/processed/

Contains transformed warehouse-ready tables.

Example:

fact_orders
dim_customers

This is your Data Warehouse layer.

notebooks/

Used for:

exploration
testing
analysis
experimentation
scripts/

Production ETL code.

extract.py

Reads CSV files.

transform.py

Cleaning + transformations.

load.py

Loads data into MySQL/PostgreSQL.

validation.py

Data quality checks.

main.py

Runs complete ETL pipeline.

sql/

Contains database scripts.

airflow/

Contains Airflow DAG automation pipeline.

dashboard/

Contains:

Power BI file
screenshots for GitHub
logs/

Stores ETL logs.

Recruiters LOVE logging systems.
